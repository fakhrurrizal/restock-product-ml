# AI Stock Service

Service backend berbasis AI untuk prediksi stok otomatis menggunakan algoritma **SARIMA** dan **Chatbot Konsultasi**.

---

## Persiapan Awal
1. **Python**: Minimal versi 3.9.
2. **Library**: Install semua kebutuhan dengan perintah:
   ```bash
   pip install -r requirements.txt
3. **Run Server**: Jalankan Server:
   ```bash
   python main.py
4. **Open Documentation**: Jalankan Server:
   ```bash
   http://localhost:8000/docs
