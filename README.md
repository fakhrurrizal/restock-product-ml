# AI Stock Service

Service backend berbasis AI untuk prediksi stok otomatis menggunakan algoritma **SARIMA** dan **Chatbot Konsultasi**.

---

## Persiapan Awal
1. **Python**: Minimal versi 3.9.
3. **Virtual Environment (Recommended)**:
   Buat dan aktifkan environment agar library terisolasi:
   ```bash
   # Membuat environment
   python -m venv venv

   # Aktivasi di Windows:
   venv\Scripts\activate

   # Aktivasi di Linux/macOS:
   source venv/bin/activate
4. **Library**: Install semua kebutuhan dengan perintah:
   ```bash
   pip install -r requirements.txt
5. **Run Server**: Jalankan Server:
   ```bash
   python main.py
6. **Open Documentation**: Jalankan Server:
   ```bash
   http://localhost:8000/docs
